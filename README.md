# github-resolve-semver
resolve an npm-style semver range to a git tag for a github repo

## usage

    var githubResolveSemver = require('github-resolve-semver')

## api


## installation

    $ npm install github-resolve-semver


## running the tests

From package root:

    $ npm install
    $ npm test
